# Module 2: Attacks, Concepts and Techniques (Cisco Cybersecurity)

## Overview
This module covered different types of cyber attacks, how they work, and the common techniques attackers use to compromise systems and networks.

---

## 2.1 Cyber Attack Basics

### Malware
Malware is malicious software designed to damage, disrupt, or gain unauthorized access to systems.

**Types of malware:**
- Virus
- Worm
- Trojan
- Ransomware
- Spyware
- Adware
- Backdoor
- Rootkit

### Symptoms of Malware
- Slow system performance
- Unexpected pop-ups
- Unknown programs running
- System crashes or unusual behavior

---

## 2.2 Methods of Infiltration

### Social Engineering
Tricking people into revealing sensitive information like passwords or access details.

### Denial-of-Service (DoS)
Overloading a system or server to make it unavailable for users.

### Distributed DoS (DDoS)
A DoS attack carried out using multiple systems (botnet) to increase impact.

### Botnet
A network of infected devices controlled by an attacker.

### On-Path Attack
An attacker secretly intercepts communication between two parties without their knowledge.

### SEO Poisoning
Manipulating search engine results to redirect users to malicious websites.

### Wi-Fi Password Cracking
Attempting to break weak Wi-Fi passwords using guessing or tools.

### Password Attacks
Common methods include:
- Brute force
- Dictionary attacks
- Credential stuffing

### Cracking Time
Stronger passwords take significantly longer to crack, depending on complexity and length.

---

## Advanced Persistent Threats (APT)
A long-term targeted cyber attack where attackers remain hidden inside a system to steal data over time.
